    • Nmap options:

    Commanly Used:

    • -sS: TCP SYN port scan
    • -sC: TCP connect port scan
    • -sV: UDP port scan
    • -O: TCP ACK port scan
    • -vv: TCP Window port scan
    • --min-rate=10000: TCP Maimon port scan
    • -p-: TCP Maimon port scan
    • -oA: TCP Maimon port scan
    • -sU: UDP port scan
    • -sA: TCP ACK port scan
    • -Pn: TCP ACK port scan

    Target Specification:

    • -iL: Scan targets from a file
    • -iR: Scan random hosts
    • --exclude: Exclude listed hosts

    Nmap Scan Techniques:

    • -sS: TCP SYN port scan
    • -sT: TCP connect port scan
    • -sU: UDP port scan
    • -sA: TCP ACK port scan
    • -sW: TCP Window port scan
    • -sM: TCP Maimon port scan

    Host Discovery:

    • -sL: List targets only
    • -sn: Disable port scanning, host discovery only
    • -Pn: Disable host discovery, port scan only
    • -PS: TCP SYN discovery
    • -PA: TCP ACK discovery
    • -PU: UDP discovery
    • -PR: ARP discovery
    • -n: Never do DNS resolution

    Port Specification:

    • -p: Port scan
    • -F: Fast port scan
    • --top-ports: Port scan the top x ports
    • -p-: Scan all ports
    • -p0-: Scan through port 65535
    • -p: Scan from service name

    Service and Version Detection:

    • -sV: Service/version detection
    • --version-intensity: Set version detection intensity
    • --version-light: Enable light mode
    • --version-all: Enable intensity level 9

    OS Detection:

    • -O: Remote OS detection
    • --osscan-limit: Limit OS detection
    • --osscan-guess: Guess OS aggressively
    • --max-os-tries: Set maximum number of OS detection tries

    Timing and Performance:

    • -T0: Paranoid IDS evasion
    • -T1: Sneaky IDS evasion
    • -T2: Polite scanning
    • -T3: Normal scanning
    • -T4: Aggressive scanning
    • -T5: Insane scanning

    Timing and Performance Switches:

    • --host-timeout: Give up on target after this long
    • --min-rtt-timeout: Specify probe round trip time
    • --min-hostgroup: Parallel host scan group sizes
    • --min-parallelism: Probe parallelization
    • --max-retries: Max number of port scan probe retransmissions
    • --min-rate: Send packets no slower than x per second
    • --max-rate: Send packets no faster than x per second

    NSE Scripts:

    • -sC: Scan with default NSE scripts
    • --script: Scan with a specific script
    • --script-args: NSE script with arguments

    Firewall / IDS Evasion and Spoofing:

    • -f: Fragmented IP packets
    • -mtu: Set own offset size
    • -D: Send scans from spoofed IPs
    • -S: Spoof source address
    • -g: Use given source port number
    • --proxies: Relay connections through proxies
    • --data-length: Append random data to sent packets

    Output:

    • -oN: Normal output to file
    • -oX: XML output to file
    • -oG: Grepable output to file
    • -oA: Output in multiple formats at once
    • -v: Increase verbosity level
    • -d: Increase debugging level
    • --reason: Display the reason a port is in a particular state
    • --open: Only show open ports
    • --packet-trace: Show all packets sent and received
    • --iflist: Show host interfaces and routes
    • --resume: Resume a scan

    Miscellaneous Nmap Flags:

    • -6: Enable IPv6 scanning
    • -h: Nmap help screen
